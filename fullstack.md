# Full Stack - Web Developer Intern Test

Tes masuk magang Fullstack Developer di PT Aksamedia Mulia Digital dirancang untuk mengevaluasi pemahaman dan kemampuan Anda dalam membuat tampilan website yang bagus, responsive, interaktif dan dinamis. Tes ini juga akan mengevaluasi pemahaman Anda terhadap website dinamis seperti pembuatan API dan integrasi API dengan frontend.

## Requirement
- Laravel
- Mysql/MariaDB
- PhpMyAdmin / Adminer / HeidiSQL
- Postman
- Html
- Css
- Tailwind
- Vue js / React js (nilai plus)

## Tugas 1
Kerjakan semua soal frontend:
[Front End - Web Developer Intern Test](https://github.com/rakafebriansy/technical-test-aksamedia/blob/master/frontend.md)

## Tugas 2
Kerjakan semua soal backend: 
[Back End - Web Developer Intern Test](https://github.com/rakafebriansy/technical-test-aksamedia/blob/master/backend.md)

## Tugas 3
Integrasikan hasil dari `Tugas 1` dan `Tugas 2` sehingga menjadi sebuah website yang dinamis

## Aturan Pengerjaan
- Ikuti arahan pada soal `frontend` dan `backend`
- Deploy hasil karya kalian dan kumpulkan link git beserta link hasil deploy
